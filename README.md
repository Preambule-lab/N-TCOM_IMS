# Deployment Instructions

To deploy the project:
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Preambule-lab/N-TCOM_IMS.git
   ```
2. Change to the project directory:
   ```bash
   cd N-TCOM_IMS
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. To run the project locally:
   ```bash
   npm start
   ```

The live webpage is hosted on GitHub Pages:
[Live Webpage](https://preambule-lab.github.io/N-TCOM_IMS/)
